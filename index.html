RapidIRC

Bukkit > IRC plugin for the Rapid Minecraft server

Download this project as a .zip file Download this project as a tar.gz file

What is RapidIRC?

RapidIRC is an in-house plugin developed by both @jakelane (aka. Manyman) and I for use on the Rapid Minecraft server. RapidIRC is designed to work with multiple servers, and the bots sit in the IRC channel and relay messages between the in-game chat and the IRC itself.

Commands

In-game commands

/irc list

Return a list of users currently in the IRC channel.

/imsg [user] [message...]

Private message a user that is in the IRC channel. Be warned, it does not check whether or not the nick is correct or if the message is correctly sent. Nicks are not case-sensitive.

IRC - Messaging

/msg [RapidSurvival/RapidCreative] [user] [message...]

Private message a player in-game through one of the server bots. The bot must be the server that the player is located in. If your IRC client opens a separate private chat window, you can then use the following to send messages, within the private chat with a bot:

[user] [message...]

IRC - Commands

~help

Return a list of commands

~players

Return a list of players online (on both worlds)

~version

Return the version of the RapidIRC plugin

Note: if you receive strange messages beginning with just - then you have been private messaged by the IRC maintenance bot, Oracle. You can see what functions Oracle has here.

Administration

In-game

Along with the in-game commands, there are a few commands that may need to be used from time to time. They can be accessed through the /ircOP command. These include:

/ircop start
/ircop restart
/ircop stop

These commands are extremely self-explanatory, and don't need much further elaboration.

In IRC

There are extra administrative commands that can perform actions in-game. 

~kick

Kick a user from the IRC channel (Requires OP in IRC)

~ban

Ban a user from the IRC channel (Requires OP in IRC)

~pardon

Un-ban/pardon a user from the IRC channel, allowing them to return (Requires OP in IRC)

Development / Plugin Administration

The plugin should create a config file in it's own folder the first time it is run with Bukkit. Within this file, the variables for the bots are stored and can be changed. Username, channel and server can be edited, along with a few other fields.

That's all you need to know about RapidIRC. Check out the Oracle bot to see what functions and features it provides on the other end.
